# Harness.pay 知识库

## 从这里开始

Harness.pay 是面向大陆/华语开发者的 MoR 平台，帮助数字产品通过卡支付和 USDC 支付出海，并由平台承接税务、退款、账本和商户结算责任。

## 当前判断

- 当前主线：先完成 Web2 MoR 闭环，再完成 Web3 USDC 闭环。
- 当前差异化：自建 Base 链 USDC 收款通道。
- 中长期方向：把 MoR 能力扩展为 Agent 交易的责任与财务执行层。
- 当前不应默认承诺：Agent 生产能力、全球全品类覆盖和开放式消费者钱包。
