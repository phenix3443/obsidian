# 业务总览

## 产品定位

Harness.pay 是面向大陆/华语开发者的 Merchant of Record（MoR）平台。平台帮助商户向欧美买家销售 SaaS、AI API、数字工具、订阅和其他数字权益，并代为处理收款、税务、发票、退款、争议、账本和 payout。

平台以 Polar 开源系统为业务基座，Web2 卡支付沿用 Stripe，Web3 通过自建通道支持 Base 链 USDC 直付。

## 目标客户

- 出海的大陆/华语开发者和独立开发者。
- AI 初创、AI API、Agent tool 和数字服务商。
- 需要多国销售、用量计费、订阅、税务和跨境结算的商户。
- 希望使用 Crypto 收款、但难以自建合规和资金流程的 Web3 开发者。

## 平台价值

对商户，平台提供一个集成面和一个责任闭环：

```text
商品/价格 -> Checkout -> 支付 -> 税务/发票 -> 权益 -> 账本 -> payout
```

对买家，平台提供统一 Checkout、订单、发票、权益和售后体验。

对平台，Polar 提供成熟的商品计费、客户权益、基础账本、Portal、API 和运营后台，平台将开发重点集中在 Crypto 支付、资金管理和合规改造。
