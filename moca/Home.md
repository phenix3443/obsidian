## Scope Rules

- `core/` 保存不受 GitHub 组织迁移影响的长期知识，面向 agent 与开发者。
- `mocachain/` 保存当前有效组织 `mocachain` 的 workspace、repo、任务和协作资料。
- 历史组织不再单独建目录，只保留迁移背景说明。

## Historical Note

> [!info]
> Moca 代码历史上曾分布在 `zkme`、`MocaFoundation`、`mocachain` 等组织下。当前有效组织是 `mocachain`；旧组织中的仓库视为归档历史，不应作为新的开发入口。
