# Organization History

> [!summary]
> Moca 代码历史上曾分布在多个 GitHub 组织下。当前有效组织是 `mocachain`，旧组织只作为历史背景保留。

## Historical Context

由于历史原因，Moca 的代码曾经放在多个组织下，包括：

- `zkme`
- `MocaFoundation`
- `mocachain`

这些组织变化属于代码托管位置的历史，不应该影响对 Moca 项目本体的理解。

## Current Rule

- 当前最新、有效的组织是 `mocachain`
- 其他历史组织中的代码视为归档
- 新的开发、排障和文档路由默认从 `mocachain` 组织开始

## Documentation Rule

- 与 Moca 项目本体有关、即使未来组织再次迁移仍然成立的知识，放在 `core/`
- 与当前 `mocachain` 组织下的 repo、workspace、任务档案有关的资料，放在 `mocachain/`
- 不再为历史组织单独建立顶层目录
