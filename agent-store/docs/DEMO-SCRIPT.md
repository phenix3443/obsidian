# Demo 视频脚本 / 分镜

目标：30–60 秒讲清「发现 → 一键装 → 立刻用上」，外加一个差异化闪光点（安全审查 / 本地代理）。
形式建议：**屏幕录制 + 字幕**（无需配音，dev 受众看字幕即可；中英各出一版最好）。
录制素材：真实商店 `agent-store-alpha.vercel.app`、桌面客户端、Claude Code。

---

## 30 秒版（主发布用，节奏快）

| 时间 | 画面 | 屏幕字幕 |
|---|---|---|
| 0–4s | Claude Code 终端里手动找 skill / 配 MCP 的凌乱截图（或快速划过几个散落的仓库标签页） | 装技能、配 MCP、接中转……太散了 |
| 4–10s | 切到 Agent Store 商店网格，鼠标滚动，停在一个 skill 卡片（如 `superpowers-brainstorming`） | 一个入口，装齐所有 Agent 能力 |
| 10–16s | 点进详情页，镜头扫过「已验证」徽章 + 「安全审查」卡片 + 「装它会执行什么」 | 每个包都经过 LLM 质量 + 安全审查 |
| 16–24s | 桌面客户端点「安装」，进度条 → 已安装；快速切到 Claude Code 里该 skill 生效 | 一键安装，立刻在 Claude Code 用上 |
| 24–30s | 落地页 + logo 定格 | agent-store-alpha.vercel.app |

## 60 秒版（PH / 官网嵌入，多一层深度）

1. **0–6s 钩子**：手动折腾的痛点（同上，稍长）。字幕：`装技能、配 MCP、接中转，一直很散。`
2. **6–16s 发现**：商店网格 → 分类切换（技能/MCP/供应商）→ 搜索 → 停在一个好包。字幕：`发现并一键安装 Claude Code / Codex 的技能、MCP 与供应商。`
3. **16–28s 信任**：详情页——tier 徽章、安全审查摘要、`将运行命令: npx -y …` 披露。字幕：`装的是 agent 会执行的东西，所以每个包都过一遍质量 + 安全审查。`
4. **28–40s 安装 + 使用**：桌面端安装 → 切 Claude Code，skill/MCP 立刻可用（展示真实效果，比如 skill 改变了 agent 行为，或 MCP 工具出现）。字幕：`一键装好，马上用上。`
5. **40–52s 差异化——本地代理**：桌面端仪表盘一瞥——多供应商、失败降级、用量/预算。字幕：`内置本地代理：跨供应商转发、自动降级，省钱又容灾。`
6. **52–60s CTA**：`一个入口，装齐所有 Agent 能力` + 链接 + 「开放贡献，欢迎提 PR」。

## 拍摄提示

- **真实操作、慢半拍**：鼠标移动放慢，关键点停 1 秒，观众才跟得上。
- **突出那三个只有我们有的画面**：安全审查徽章、「会执行什么」披露、本地代理仪表盘——这是差异化，多给镜头。
- **首帧要能当封面/OG**：商店网格或详情页的信任徽章，信息量足。
- **配一段无版权 BGM**，节奏对齐剪辑点。
- 导出 **1080p**，社交平台再压；同时截 3–5 张静图做首图/推文配图。
